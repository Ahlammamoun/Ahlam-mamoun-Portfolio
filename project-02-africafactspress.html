Média symfony react mysql docker ci/cd nginx

Fiche projet

Type

Site média : actualités, enquêtes et analyses sur l'actualité africaine.

Focus

Organisation du contenu, navigation par catégories et expérience de lecture optimisée.

Mon rôle

Développement fullstack : API Symfony, interface React, gestion de la base MySQL et déploiement Docker avec pipeline CI/CD via GitHub Actions.

Stack

Front: React
Back: Symfony (API) + Doctrine
Paiement: Stripe
DB: MySQL
Infra: Docker + Nginx
CI/CD: GitHub Actions

Fonctionnalités

  • Publication d’articles avec catégories et rubriques
  • Page d’accueil dynamique avec articles récents et contenus mis en avant
  • Navigation par thématiques (actualité, analyses, enquêtes)
  • Optimisation SEO : meta tags, Open Graph, structure d’articles
  • Interface d’administration pour gérer les contenus (création / modification / suppression)

Points techniques

  • API développée avec Symfony et Doctrine
  • Interface utilisateur en React
  • Base de données MySQL pour la gestion des contenus
  • Environnement Docker avec Nginx pour le serveur web
  • Pipeline CI/CD automatisé avec GitHub Actions
