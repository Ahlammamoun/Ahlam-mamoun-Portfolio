Bing-Bang
Mini jeu HTML / CSS / JS

Petit jeu web en JavaScript : tu déplaces le personnage au clavier (← →) pour éviter les obstacles, avec génération dynamique et score.

🎮 Jouer (GitHub Pages) 💻 Code GitHub ← Retour aux projets
JavaScript DOM Game loop GitHub Pages

Fiche projet

But

Survivre en se déplaçant à gauche/droite pour passer dans les ouvertures.

Contrôles

Clavier : flèches ← → (gestion keydown/keyup).

Score

Calcul du score et affichage en fin de partie, puis relance.

Stack

HTML • CSS • JavaScript
Déploiement : GitHub Pages

Gameplay

  • Déplacement horizontal du personnage via listeners clavier
  • Obstacles générés dynamiquement dans le DOM, avec ouverture aléatoire
  • Défilement des obstacles via une boucle de mise à jour
  • Fin de partie : affichage du score + relance

Points techniques

  • Manipulation DOM (createElement, appendChild, remove)
  • Lecture de styles/positions pour gérer les collisions
  • Timers (setInterval) pour l’animation et la génération
  • Déploiement sur GitHub Pages

Améliorations possibles (version 2)

Support mobile (touch), score en UI (sans alert), difficulté progressive, écran start/retry, et boucle plus fluide avec requestAnimationFrame.
