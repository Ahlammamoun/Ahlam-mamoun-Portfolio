E-commerce produits de crochet

Boutique en ligne dédiée aux produits de crochet : catalogue, panier et paiement sécurisé via Stripe, avec une architecture moderne React + Symfony.

🛍️ Voir le site ← Retour aux projets → Projet suivant
React Symfony MySQL Stripe Docker Nginx GitHub Actions

Fiche projet

Type

Site e-commerce (produits crochet) : catalogue, panier, commande.

Mon rôle

Développement fullstack : front React, API Symfony, intégration Stripe, DB MySQL et déploiement Docker.

Focus

Expérience d’achat simple, paiement sécurisé et back-office pour gérer les produits.

Stack

Front: React
Back: Symfony (API) + Doctrine
DB: MySQL
Paiement: Stripe
Infra: Docker + Nginx
CI/CD: GitHub Actions

Fonctionnalités

  • Catalogue produits (listes + fiches produit)
  • Panier : ajout/suppression, quantités, total
  • Commande + paiement sécurisé via Stripe
  • Gestion des produits (création / modification / suppression) via admin
  • Gestion des commandes (suivi, statut) (si activé)

Points techniques

  • API Symfony structurée + Doctrine (entités, migrations)
  • Intégration Stripe (checkout + webhooks) pour fiabiliser les paiements
  • MySQL : produits, catégories, commandes, utilisateurs
  • Docker : environnement reproductible + Nginx reverse proxy
  • CI/CD GitHub Actions : build/tests + déploiement automatisé

Ce que j’ai appris

Mettre en place un e-commerce complet (catalogue → panier → paiement), sécuriser le parcours et fiabiliser Stripe via webhooks, tout en industrialisant le déploiement avec Docker + CI/CD.
