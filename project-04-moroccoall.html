MoroccoAll
E-commerce artisanat marocain

Boutique en ligne mettant en valeur des produits d’artisanat marocain : catalogue, catégories, fiches produit et paiement sécurisé via Stripe.

🧺 Voir le site ← Retour aux projets → Projet suivant
React Symfony MySQL Stripe Docker Nginx GitHub Actions

Fiche projet

Type

E-commerce : catégories, fiches produit, panier, commande.

Mon rôle

Développement fullstack : React côté front, API Symfony, DB MySQL, intégration Stripe et déploiement Docker.

Focus

Mise en valeur des produits, parcours d’achat simple et paiement sécurisé.

Stack

Front: React
Back: Symfony (API) + Doctrine
DB: MySQL
Paiement: Stripe
Infra: Docker + Nginx
CI/CD: GitHub Actions

Fonctionnalités

  • Navigation par catégories + recherche (si présente)
  • Fiches produits : description, prix, images, stock
  • Panier : quantités, total, suppression
  • Commande + paiement sécurisé via Stripe
  • Administration : gestion produits/catégories (CRUD) (si activé)

Points techniques

  • API Symfony structurée + Doctrine (entités, migrations)
  • Gestion des images produits + optimisation (poids / formats)
  • Stripe : checkout + webhooks (fiabiliser le statut paiement)
  • Sécurité : validation des entrées, contrôle d’accès admin
  • Docker + Nginx : environnement stable et reproductible
  • CI/CD GitHub Actions : build/tests + déploiement

Ce que j’ai appris

Mettre en place un parcours e-commerce complet, gérer un catalogue (CRUD admin), intégrer Stripe proprement (webhooks) et industrialiser le déploiement avec Docker + CI/CD.
